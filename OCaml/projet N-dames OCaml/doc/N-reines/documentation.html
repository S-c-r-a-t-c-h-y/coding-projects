Documentation du code

Ci-dessous la documentation pour toutes les fonctions du code.

type reine = int * int

une reine est représentée par un couple de coordonées (i, j)

val cases_prises : reine list -> int -> int -> int array -> int array

cases_prises reines ligne n arr calcul les cases de la ligne ligne pouvant être attaquées par une des reines de reines déjà placées

  • parameter reines

    une liste des coordonnées des reines déjà placées sur le plateau

  • parameter ligne

    ligne pour laquelle on souhaite obtenir les indices des cases prises

  • parameter n

    nombre de reines

  • parameter arr

    array contenant n nombres, tous initialement à 0

  • returns

    un array de n nombres valant 1 si une reine de reines peut attaquer la case à la position (i, ligne) où i est l'indice dans l'array, et 0 sinon.

val indices_valides : reine list -> int -> int -> int array

indices_valides reines ligne n calcul les indices des cases sur lesquels on peut poser une reine sur la ligne ligne sans que celle ci ne puisse être attaquée par une reine de reines

  • parameter reines

    une liste des coordonnées des reines déjà placées sur le plateau

  • parameter ligne

    ligne pour laquelle on souhaite obtenir les indices valides

  • parameter n

    nombre de reines

  • returns

    un array de tous ces indices

Résolution pour UNE solution (méthode classique)

val solution : int -> reine list

solution n calcul une solution au problème des n-reines pour un plateau de n*n.

  • parameter n

    taille de l'échiquier

  • returns

    une liste contenant les coordonnées des n reines

val print_solution : int -> Stdlib.out_channel -> unit

print_solution n affiche dans out_chan le plateau d'échec de n*n avec les n reines placées au bon endroit.

  • parameter n

    taille de l'échiquier

  • parameter out_chan

    channel de sortie sur lequel doit être affiché la solution (stdout ou un fichier)

Résolution pour TOUTES les solutions

val remove_isomorphismes : reine list list -> reine list list

remove_isomorphismes sols supprime toutes les solutions isomorphes à d'autres solutions, ne laissant que les solutions de "base"

  • parameter sols

    la liste de toutes les solutions

  • returns

    la liste des solutions uniques à isomorphisme près

val solutions : int -> bool -> reine list list

solutions n unique renvoie une liste de toutes les solutions pour un plateau de n*n, uniques à isomorphisme près si unique=true.

  • parameter n

    taille de l'échiquier

  • parameter unique

    si oui ou non les solutions renvoyées sont uniques

val print_nb_solutions : int -> bool -> Stdlib.out_channel -> unit

print_nb_solutions n unique out_chan affiche dans out_chan le nombre de solutions pour un plateau de n*n, uniques à isomorphisme près si unique=true.

  • parameter n

    taille de l'échiquier

  • parameter unique

    si oui ou non les solutions affichées sont uniques

  • parameter out_chan

    channel de sortie sur lequel doit être affiché la solution (stdout ou un fichier)

val print_solutions : int -> bool -> Stdlib.out_channel -> unit

print_solutions n unique out_chan affiche dans out_chan toutes les configurations possibles de n reines sur un plateau de n*n, uniques à isomorphisme près si unique=true.

  • parameter n

    taille de l'échiquier

  • parameter unique

    si oui ou non les solutions affichées sont uniques

  • parameter out_chan

    channel de sortie sur lequel doit être affiché la solution (stdout ou un fichier)

Résolution pour UNE solution (méthode de résolution de conflits)

val place_n_reines : int -> reine list

place_n_reines n place n reines aléatoirement sur un plateau de n*n

  • parameter n

    nombre de reines à placer

  • returns

    une liste des n reines

val cases_conflit : (int * int) -> int -> (int * int) list

cases_conflit (x, y) n calcul les coordonnées de toutes les cases attaquées par une reine placée aux coordonées (x, y)

  • parameter (x,

    y) les coordonnées x et y de la case dont on souhaite connaître le nombre de conflit

  • parameter n

    la taille tu tableau

  • returns

    une liste de toutes les coordonnées attaquées par la reine

val calculer_conflits : int -> reine list -> int * bool * (int * int) list

calculer_conflits n reines calcul les conflits causés par les reines afin de déterminer la reine avec le plus de conflits

  • parameter n

    la taille de l'échiquier

  • parameter reines

    les positions des reines

  • returns

    l'indice de la reine ayant le plus de conflit, un booléen valant vrai si et seulement si aucun reine n'est dans un conflit et les coordonées des cases en conflit

val resoudre_conflits : int -> reine list -> reine list

resoudre_conflits n reines résoud les conflits entre les reines s'il y en a, et recalcul un nouveau set de reines si un extremum local est atteint

  • parameter n

    la taille de l'échiquier

  • parameter reines

    les coordonnées des reines

  • returns

    une liste des n reines placées sans conflit

val solution_conflits : int -> reine list

solution_conflits n résoud le problème des n-reines en utilisant une méthode de résolution de conflits

  • parameter n

    la taille de l'échiquier

  • returns

    une liste des coordonnées des n reines

val print_solution_conflits : int -> Stdlib.out_channel -> unit

print_solution_conflits n affiche dans out_chan une solution au problème des n-reines obtenue à l'aide d'un algorithme de résolution de conflits

  • parameter n

    la taille de l'échiquier

  • parameter out_chan

    channel de sortie sur lequel doit être affiché la solution (stdout ou un fichier)
